Exploring Technology Trends in Data Journalism: A Content Analysis of CAR Conference Session Descriptions

I conceived and executed this Information Science Senior Project during September-December 2019 at Northeastern University. I wanted to combine the two parts of my major, information science and journalism, and study the field in which I want to work. Using Python and Excel, I implemented keyword discovery to assess popularity of technologies in the data journalism domain based on topics discussed at a major conference. The result is an analysis of trends from the last 5 years, leading to recommendations for students and practitioners based on the current technology landscape.

Finding Your Way | GitHub repo

This group project for my Human-Computer Interaction class is a redesign of Northeastern University's degree audit system. The current system, used to see a student's degree progress and explore other possible degrees, is unintuitive for users because all the information is presented at once and there is no visual hierarchy. My group executed the complete design process including proposal, requirements gathering, sketches, paper prototyping, soliciting feedback to computer prototyping using React. I'm especially proud of implementing the "Explore" page using react-select and react-tooltip.

Boston's Closed Port | GitHub repo

My classmates Danae Bucci, Jasmine Heyward, Jasmine Wu, and I conceived and executed this project for the journalism course Coding for Digital Storytelling at Northeastern University during January-April 2019. I personally wrote the introduction and coded the attractions and transportation maps. I'm especially proud of how the transportation map turned out with the animations and scrollytelling (i.e., routes appear when you scroll to the relevant paragraph in the written narrative). I only had minimal JavaScript experience prior, and this was the first time I used the D3 and Leaflet libraries. Most of the actual code for the animations came from this extremely helpful tutorial, which I tweaked for our needs. Through this project, I learned a lot about web development and integrating interactive elements using HTML/CSS/JavaScript elements to tell a cohesive story.
