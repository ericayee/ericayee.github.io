Boston's Closed Port | GitHub repo

My classmates Danae Bucci, Jasmine Heyward, Jasmine Wu, and I conceived and executed this project for the journalism course Coding for Digital Storytelling at Northeastern University during January - April 2019. I personally wrote the introduction and coded the attractions and transportation maps. I'm especially proud of how the transportation map turned out with the animations and scrollytelling (i.e., routes appear when you scroll to the relevant paragraph in the written narrative). I only had minimal JavaScript experience prior, and this was the first time I used the D3 and Leaflet libraries. Most of the actual code for the animations came from this extremely helpful tutorial, which I tweaked for our needs. Through this project, I learned a lot about web development and integrating interactive elements using HTML/CSS/JavaScript elements to tell a cohesive story.

In my Spring 2019 Bostonography course, I learned how to apply computational methods to cultural, historical, and society questions about Boston. I became well-versed in using R for data cleaning and visualization through the assignments for this course, especially the tidyverse and ggplot libraries. Here are a few of my Rmarkdown projects:

Analyzing #GoldOpen Tweets with R | RMarkdown

For my first foray into R, I adapted this Storybench tutorial about mapping NBA finals tweets to look at #GoldOpen tweets using the rtweet package. That hashtag was used in a social media campaign to ensure a successful opening for the film "Crazy Rich Asians." Take a look at the markdown file linked above for how I graphed the tweets over time and mapped them.

Developing a Composite Measure: Restfulness | Jupyter Notebook

For my Spring 2019 course Empirical Research Methods, I developed a survey about restfulness and distributed it to my classmates. I analyzed the resulting data using Python and the pandas, matploblit, and scipy libraries.
